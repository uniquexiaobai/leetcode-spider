[![NPM](https://img.shields.io/npm/v/@lokibai/leetcode-spider.svg)](https://www.npmjs.com/package/@lokibai/leetcode-spider)

```bash
npm i @lokibai/leetcode-spider
```

## Usage

`.leetcode.json`
```json
{
	"username": "xxx",
	"password": "xxx",
	"dist": "./docs/solutions"
}
```

